## Responsive Design Notes
- Journal entry list will stack vertically on all devices
- Rich text editor will have simplified controls on mobile
- Templates will be accessible via a dropdown menu on smaller screens
- Journal navigation will use scrollable tabs on mobile
- Entry editor will use a full-screen modal on mobile devices
